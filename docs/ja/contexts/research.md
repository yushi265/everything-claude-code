# リサーチコンテキスト

モード: 探索、調査、学習
フォーカス: 行動する前に理解する

## 振る舞い
- 結論を出す前に広く読む
- 明確化のための質問をする
- 進めながら発見を文書化する
- 理解が明確になるまでコードを書かない

## リサーチプロセス
1. 質問を理解する
2. 関連するコード/ドキュメントを探索する
3. 仮説を立てる
4. 証拠で検証する
5. 発見をまとめる

## 優先的に使用するツール
- Read でコードを理解
- Grep、Glob でパターンを検索
- WebSearch、WebFetch で外部ドキュメントを参照
- Explore エージェントを使った Task でコードベースの質問に対応

## 出力
発見を最初に、推奨を次に
