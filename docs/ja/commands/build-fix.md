# ビルドと修正

TypeScriptとビルドエラーを段階的に修正します：

1. ビルドを実行: npm run build または pnpm build

2. エラー出力を解析:
   - ファイルごとにグループ化
   - 重要度順にソート

3. 各エラーに対して:
   - エラーのコンテキストを表示（前後5行）
   - 問題を説明
   - 修正を提案
   - 修正を適用
   - ビルドを再実行
   - エラーが解決したか確認

4. 以下の場合は停止:
   - 修正により新しいエラーが発生
   - 同じエラーが3回の試行後も継続
   - ユーザーが一時停止を要求

5. サマリーを表示:
   - 修正したエラー
   - 残りのエラー
   - 発生した新しいエラー

安全のため、一度に1つずつエラーを修正します！
